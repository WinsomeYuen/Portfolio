Surplus

This was a 3 month project created for my software engineering module at university working in a group of 5. I was the Project leader for the group which consisted of:

  • - Finalising the designs and requirements for the application
  • - Ensuring all requirements were met during development
  • - Taking on a full-stack developer role
  • - Keeping track of team member's progress and assisting if help was required
  • - Deciding on the technology to use on the project based on team members skills and opinions

The website's aim was to reduce food waste caused by restaurants and supermarkets, by offering a platform that would benefit both the customers and businesses, by creating a platform which allowed businesses to sell products that were close to expiration, undesirable, over-stocked for a cheaper price to customers. To do this the web application would store products for each businesses in a database, that would be retrieved when customers clicked on a location marker on a interactive map to lok at the products available. Customers can add products from different businesses and the Stripe API is used to process payments. Past orders would be stored for each customer in a database that stored their logins.
