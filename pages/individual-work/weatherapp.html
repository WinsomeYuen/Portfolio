Water Watch

This was a 3 month project created for my graphical user interface module at university working in a group of 5. I was the Lead Developer for the group. Below are the initial design mockups/wireframes we came up with when researching stakeholders, coming up with requirements and planning the roadmap.

The weather app was to be used by people who partake in watersports. The user would type in a location (country, city, postcode) which would then be used by the geolocation api to get the co-ordinates, that would then query the weather api to get a json file with weather information that can be displayed about the location.
