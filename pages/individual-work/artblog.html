Art Blog

The aim of the project was to make a blog of your choice that had a login and allowed users to make posts. I added the MySQL database as an extra to the project to reduce the number of files stored locally. The project taught me how to add functionality to a static website using php and javascript, to add a creative flair to the blog. As the blog was a quick project, there wasn't a lot of to do for the design stage, but I did map out the content and pages required for the blog, and made a test website to learn to attach a MySQL database onto a website.

This was one of my first proper projects and it introduced me to using php, which is one of my favourite languages to use for web development nowadays.
