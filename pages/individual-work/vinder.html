Vinder

A Django website that allows people to find others who have similar hobbies. This is a web programming 3rd year group project. We had less than a month to make this website from scratch. We had a basic outline of requirements for the website, and the rest was up to us to make the website look fancy. The project lead me to learn how to deploy a website on AWS, (was probably the hardest bit) which will definitely be useful for the future!
